# spring-cloud

### 개요
Spring Cloud로 개발하는 마이크로서비스 애플리케이션(MSA) 강의를 듣고 필요한 내용을 정리하거나, 나만의 방식으로 구현하고 싶은 코드를 관리하는 저장소 입니다.

### 기타
강의에서는 maven 환경과 각 애플리케이션을 별도로 생성하지만 해당 레파지토리에서는 애플리케이션을 별도로 나누지 않고 멀티모듈 형태로 구축하며, gradle을 사용합니다.
